**Desafio Criando Herói** -  Projeto de estudos javascript proposto para praticar os conceitos de :

- Variáveis
- Operadores
- Laços de repetição
- Estruturas de decisões
- Funções
- Classe
- 
Neste projeto temos como objetivo criar um heroi, sendo assim usamos uma Classe para representa-lo e torna facil e reutilizavel.
